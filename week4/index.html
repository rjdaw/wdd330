    Forms

    The following line of code would get the first form in the document with the [0]. A similar line with forms[1] would get the second form.

    const form = document.forms[0];

    We can also look for a form by using its name the following way.

    const form = document.forms['hero'];

    Object Oriented Programming in JavaScript

    Here is a way to create a constructor.

    The "this" keyword indicated that the number of sides of the object by default is whatever is going to be passed in. In this case "6".

    The roll function here is actually a method that is being created and that can be used by any instance of a dice from now on.

    const Dice = function(sides=6){
    this.sides = sides;
    this.roll = function() {
        return Math.floor(this.sides * Math.random() + 1)
    }
}

    If I wanted to create a 20-sided dice, I would do it the following way.

    const twentySided = new Dice(20);

    A more modern way to to this would be to use the class keyword and create a constructor function like so:

    class Dice {
    constructor(sides=20) {
        this.sides = sides;
    }
    roll() {
        return Math.floor(this.sides * Math.random() + 1)
    }
}
