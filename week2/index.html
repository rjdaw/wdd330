    Arrays

    I can declare an array as an object just like in Java.
    const myArray = new Array();

    I can swap two values from an array by doing something like this:
    [x,y] = [y,x];

    The shift() method pops the first item of the array, while the unshift() method adds an item at the start.
    prophets.shift(); pops out Joseph Smith.
    prophets.unshift('Joseph Smith'); would put him back at the start.

    concat merges the arrays
    prophets.concat(['Gordon B. Hinckley', 'Thomas S. Monson', 'Russell M. Nelson']); would add this array after the already existing prophets array

    slice takes a part of the array
    prophets.slice(3, 5); would give us the prophets starting at index 3 (4th prophet: Wilford Woodruff) and finish at the fifth position (Lorenzo Snow). In this case, it would return those two prophets.

    splice replaces parts of the array
    prophets.splice(0, 1, 'Brigham Young') would start deleting at index 0, delete 1 item, replace it with Brigham Young
    Note that if 0 item is removed, the item is simply inserted at the referenced index
