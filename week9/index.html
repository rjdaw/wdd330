    The Window Object

    window.something

    The window object is basically a global object.

    alerts, confirms, and prompts are all derived from the window object.

    window.navigator gives information on the browser used by the user. It would give things such as the name of the browser, the language, if cookies are enables, etc.

    window.innerWidth & window.innerHeight tell you how big your window is.

    window.location gives info on the url you are on.

    Since window is an object. It can be called as so, meaning we can look for variables it contains. For example:

    • window.location.hostname
    • window.location.port
    • window.location.href
    • window.location.search

    window.history gives information about previously visited pages.

    window.history.length for example, gives you the number of pages visited this session.

    window.screen has more uses for mobile devices. It also allows you to do things like turn off the deviceâ€™s screen, detect a change in its orientation or lock it in a specific orientation.

    document.something

    document.write() shouldn't be used anymore to output things to the screen.

    document.cookie can be used to save some pieces of information from one page to another.

    window.setTimeout() can be used to invoke a function after a set amount of time. It can be canceled using clearTimeout().

    window.setInterval() can be used to repeatedly invoke a function. It can be stopped using clearInterval().

    HTML5 APIs

    data- attributes

    data- attributes are a way of embedding data in a web page using custom attributes that are ignored by the browser.

    One of those uses is the lazy loading for example.

    APIs

    The Web Storage API works like the cookies in PHP with key-value pairs.

    The Geolocation API gets the location of the user's device.

    The Web Worker API makes heavier tasks or tasks that could take longer to be run in the background to avoid unresponsive pages or other problems.

    There is a Notification API that displays notifications.

    The Media API along with the <audio> and <video> elements allow you to embed audios and videos in web pages.
