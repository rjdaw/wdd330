    Local Storage

    • localStorage.clear() to clear stored data
    • localStorage.setItem("name", "Ruben") sets the key as "name" and the value as "Ruben"
    • localStorage.removeItem("name") would remove the item that has the passed in key
    • localStorage.getItem("age") would allow us to get the value that matches the age key.
    • localStorage.key(1) would give us the key at index[1]

    Example of Local Storage application

    //Display local storage
    const inputKey = document.getElementById("inputKey");
    const inputValue = document.getElementById("inputValue");
    const btnInsert = document.getElementById("btnInsert");
    const lsOutput = document.getElementById("lsOutput");

    btnInsert.onclick = function () {
      const key = inputKey.value;
      const value = inputValue.value;

      if (key && value) {
        localStorage.setItem(key, value);
        location.reload();
      }
    };

    for (let i = 0; i < localStorage.length; i++) {
      const key = localStorage.key(i);
      const value = localStorage.getItem(key);

      lsOutput.innerHTML += `${key}: ${value}< br>`;
    }

    Master Mobile UX - Lesson 1

    Step 1

    Trying to put a marshmallow into a piggy bank is neither simple nor beautiful.
    That is why we must design and develop the other way around, starting with the smallest device.
    Whatever you are doing DO MOBILE FIRST!

    Step 2

    1. Be present
    2. Be useful
    3. Be quick

    Things at the top of the screen tend to be harder to reach

    Step 3

    What if we put the nav bar at the bottom of the screen?
    It would make it easier for the user to reach.

    The word "menu" is better than a hamburger.
    There is a better way to do things.

    Step 4

    • Make the user think less!
    • Display labels as placeholders.
    • Make the placeholder float above what the user writes so that he doesn't forget.
    • Increase height of fields to make it easier to tap.
    • Make a progress bar so the user knows how much more to do.
    • Enable autocomplete.
    • Show the password and offer the possibility to hide.
    • Bring in number pad when entering a number (e.g. phone number)

    Step 5

    Ads can be small and get bigger when you click on them instead of huge images that hurt readability.

    A genius idea would be to have a window between content that shows full ad as you scroll.

    Stop using pop-ups and pop-vers

    Content first!
