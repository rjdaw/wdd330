    Arrow functions have no "this"

    Objects are copied by reference

    For example:

        const thor = { name: 'Thor' 
    };
    const cloneThor = thor;
     
    cloneThor.name = 'Clor';
   
    thor.name
    // prints 'Clor'

    The Document Object Model (DOM)

    You can use querySelector on :first-child, last child, etc.

    const wonderWoman = document.querySelector('li:last-child');

    Example of creating an element:

        function createElement (tag,text) {
    const el = document.createElement(tag);
        el.textContent = text;
        return el
    }
           
    const aquaman = createElement('li','Aquaman');

    Example of updating the CSS:

        superman.style.border = "red 2px solid";

    properties that include a '-' like background-color for example are camelCased. For example:

    superman.style.backgroundColor = 'blue';

    The classList property adds a class to an element.

    wonderWoman.classList.add('warrior');

    Events

    Example of adding an event listener

    function doSomething() {
    alert('You Clicked!');
}
addEventListener('click', doSomething);

    It also works with arrow functions

    addEventListener('click', () => alert('You Clicked!'));

    There are lots of different events, including actions done with:

    • A mouse
    • A keyboard
    • A finger (touching the screen)

    You can remove an event using the removeEventListener method.
